The Data: Make a CSV file with "Category" and "Amount." Use your four groups: Rent, Food, Utilities, and Discretionary.

The Reader: Write Python code to open that file and add up the total money spent in each group.

The Math: Group those totals into Needs, Wants, and Savings. Figure out what percentage of your income each one is.

The Result: Print those percentages to the screen so you can see if you hit the 50/30/20 goal.
